HTML5 Test Game

Learning HTML5 and advanced javascript through game development.

Part 1 - Physics Engine

  1. Momentum and Drag, v.1
    momentum_and_drag.html
    Use the arrow keys to move the block around the screen. Fake thrust and friction apply.
  2. Simple Collisions
    simple_collision.html
    Single-dimension movement, collision detection and resolution
  3. Billiard Physics, v.1
    billiards_physics.html
    Objects moving in 2-dimensions with different masses, velocities, and radii, with enhanced collision detection and resolution.
  4. Vector Math
    vector_math.html
    Creating a library of vector functions
  5. Wall Physics
    wall_physics.html
    A single object rebounding off of a wall.
  6. Momentum and Drag, v.2
    momentum_and_drag2.html
    Same as momentum_and_drag.html, but now using actual acceleration, friction, and mass.
  7. Momentum and Drag v.3 - Pilot
    pilot.html
    Enhanced version of momentum_and_drag2.html that now uses a ship with rotation and thrust, instead of cartesian movement. Also introducing object inheritance
  8. Billiards Physics, v.2
    billiards_phyics2.html
    Billiards physics plus the momentum and drag logic, plus collision dampening
  9. Wall Pysics, v.2 (IN PROGRESS)
    wall_physics2.html Solid, filled walls with multiple line segments and collision dampening.
  10. Physics Engine, v.1
    Combined billiard physics, wall physics, and pilot. Better class organization for objects (pilot, walls, agents each have collision detection and resolution algorithms)
  11. Physics Engine, v.2
    A more efficient engine with bounding boxes for predicting collisions and cleaned up math.
